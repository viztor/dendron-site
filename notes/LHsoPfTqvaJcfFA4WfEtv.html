NextJS Publishing

Dendron Next Generation Publishing

Prerequisites

  1. Install latest version of Dendron CLI
  2. Install yarn (optional)

Gotchas

  • Dendron Next publishing uses pretty urls which means pages will no longer have a .html suffix. If you are migrating an existing 11ty based Dendron site to Next, make sure to redirect .html pages to preserve existing links

Process

Setup

Run the following commands at the root of your dendron workspace

From Common
Go to text â†’
  1. Update .gitignore 
    echo .next >> .gitignore
  2. Clone the nextjs repository 
    dendron publish init
  3. Install dependencies 
    cd .next && yarn && cd ..

Build Notes

  • NOTE: run this command inside your workspace
dendron publish build

Preview Notes

  • NOTE: run this command inside your workspace
  • visit http://localhost:3000 to see your local preview
cd .next && yarn dev && cd ..

Export Notes

  • NOTE: run this command inside your workspace
cd .next && yarn export

Preview
